How to start JupyterLab in AWS Cloud9 ************************************** Create Python based virtual environment - python3 -m venv dataeng_lab Activate virtual environment - source dataeng_lab/bin/activate Install required dependencies for Jupyter Lab - pip install jupyterlab Launch Jupyter Lab - jupyter lab --ip 0.0.0.0 At this time, you will not be able to access Jupyter Lab Go to firewall and open the port using AWS Web Console jupyter lab --port $PORT --no-browser ==> can be used as well
